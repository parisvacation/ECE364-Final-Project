ECE364 Final Project

The final project is designed to give you hands-on experience building and improving machine learning models on real-world tasks. We offer three independent tracks—you choose one and explore any creative or unconventional methods you can think of to beat the baseline. This is an open-ended challenge, and the most important thing we want to see is effort. Details for Spring 2026 have been released!

Project Tracks

Each track provides a baseline model, a dataset, and a clear evaluation metric. Select one and improve upon the baseline. If you have any project-related questions, feel free to open an issue on GitHub or contact the TA.

1

3D Point Cloud Classification

Build a robust classifier for corrupted 3D point clouds. Given a point cloud that has been perturbed by one of several synthetic corruption types, your model must predict the correct object category among 40 classes.

The dataset is based on ModelNet40. Each source shape is replicated with random corruptions (jitter, scale, rotation, global/local dropout, outlier addition, density increase) at three severity levels, yielding 25,086 training, 4,434 validation, and 7,404 test samples. Test files are anonymised so that class labels are not available during inference.

The baseline is a from-scratch PointNet++ (SSG) with 1.47M trainable parameters, achieving 86.60 % test accuracy without additional augmentation.

2

AutoResearch

Write an agent (agent.py) that automatically searches for a better training pipeline. The agent reads a baseline train.py, iteratively asks an LLM (Kimi-K2.5) to propose improved versions, validates each proposal locally, and keeps only candidates that improve validation accuracy.

The evaluation benchmark is an intent classification task with 64 categories, containing 7,870 training, 1,084 validation, and 1,076 test examples. A separate development benchmark (35 categories) is provided for local debugging.

The baseline model is a Deep Averaging Network (DAN) with learnable embeddings. The agent runs in a sandboxed CPU-only environment with a budget of 10 iterations and a total time limit of 2,400 seconds.

3

Cigarette Constituent Prediction

Predict the amounts of three harmful constituents—Tar (mg/cigarette), Carbon Monoxide (mg/cigarette), and Phenol (µg/cigarette)—from 36 physical and chemical features of cigarette samples.

The training set contains 695 samples with all three target values, though Phenol has 390 missing entries that require careful handling. The test set comprises 190 samples with no target labels provided.

The baseline is a TransformedTargetRegressor with log1p transform and XGBoost for each target independently, combined with custom puff-feature engineering and 5-fold cross-validation. It achieves an average R² of 0.4018 across the three targets.
