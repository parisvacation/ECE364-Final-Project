ECE364 Final Project

Three independent tracks covering 3D point cloud classification, automated machine-learning research with large language models, and regression-based constituent prediction. Choose one track, build on the provided baseline, and submit your best results.

Project Tracks

Each track provides a baseline model, a dataset, and a clear evaluation metric. Select one and improve upon the baseline.

1

3D Point Cloud Classification

In this track you build a robust classifier for corrupted 3D point clouds. The dataset is based on ModelNet40 and contains 40 object categories with seven types of synthetic corruption—jitter, scale, rotation, global dropout, local dropout, global outlier addition, and density increase—at three severity levels. The training set provides 25,086 labelled point clouds and the validation set 4,434, while the test set (7,404 samples) uses anonymised file names so that class labels are not available during inference.

The reference baseline is a from-scratch PointNet++ (SSG) implementation with 1.47 million trainable parameters. Without any additional data augmentation beyond the on-disk corruptions, it achieves 86.60 % classification accuracy on the held-out test set. Your goal is to improve upon this accuracy through better architectures, training strategies, or data processing.

2

AutoResearch

Rather than manually tuning a model, this track asks you to write an agent (agent.py) that automatically searches for a better training pipeline. The agent reads a baseline train.py—a Deep Averaging Network (DAN) for intent classification—and iteratively asks an LLM (Kimi-K2.5) to propose improved versions. Each proposal is validated locally; only candidates that improve on the current best validation accuracy are kept.

The evaluation benchmark contains 64 intent categories with 7,870 training, 1,084 validation, and 1,076 test examples. The agent is executed in a sandboxed CPU-only environment with a total time limit of 2,400 seconds and a budget of 10 search iterations. Final grading reads best_test_accuracy from the agent's output manifest.

3

Cigarette Constituent Prediction

This is a regression task: given 36 physical and chemical features of a cigarette sample, predict the amounts of three harmful constituents—Tar (mg/cigarette), Carbon Monoxide (mg/cigarette), and Phenol (µg/cigarette). The training set contains 695 samples, though Phenol has 390 missing values that require careful handling. The test set has 190 samples with no target labels.

The baseline pipeline uses a TransformedTargetRegressor with log1p transform and XGBoost for each target independently, combined with custom puff-feature engineering and 5-fold cross-validation. It achieves an average R² of 0.4018 across the three targets. Your submission is evaluated by computing the R² score for each target and taking the mean.
